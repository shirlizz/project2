## Usage

* Choose a username
* Create a new channel or select an existent one

## How to run

```bash
# Install all dependencies
$ pip install -r requirements.txt

# Export application
$ export FLASK_APP=application

# Run
$ flask run

# Go to 127.0.0.1:5000 on your web browser.
```
